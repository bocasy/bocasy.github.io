Textile uncoiler machine

In this project we created a machine that uncoiles rolls of cloth. It is used as part of an textile technological line, it's the first component that feeds material at a constant speed.

Motor presentation

Rolls that are uncoiled are heavy and we went with a big motor to be sure this machine can handle a large variety of types of dimensions.
